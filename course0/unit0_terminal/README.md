# Unit 0: Terminal/CLI Fundamentals

Learn to navigate your filesystem using the command line. You'll use pwd, ls, cd, mkdir, touch, and cp to move around, create directories, create files, and copy files.

**Estimated Time:** 10 minutes

## How to Validate

These are hands-on terminal exercises. You'll know you got it right when the commands work and produce the expected results. No grader for this unit - just follow the instructions and verify the output yourself.

Take the quiz (`quiz.yaml`) for self-reflection.
